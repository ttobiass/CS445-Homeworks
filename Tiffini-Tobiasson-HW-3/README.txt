Prerequisites: 
openJDK-8-jdk
JUnit4

To aquire these prerequisites run the following commands: 
sudo apt-get update
sudo apt-get install openjdk-8-jdk
sudo apt-get install junit4

Build Instructions: 
In the current directory run:
make clean
make

Run Instructions: 
In the current directory run:
make run

Run Unit Tests Instructions:
In current directory run: 
make unittest
